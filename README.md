## ЁЯЪА Getting Started

### Prerequisites

- Node.js `>= 18`
- npm

### Local Development Setup

```bash
# Clone the repository
git clone https://github.com/kutluhangil/My-Journey-Portfolio.git
cd My-Journey-Portfolio

# Install dependencies
npm install

# Start the Vite development server (runs typically on http://localhost:5173)
npm run dev
```

## тШБя╕П Deploy to Vercel

The project is natively routed and structured for immediate Vercel deployments. The root directory contains a `vercel.json` and a specially configured `vite.config.ts`.

### Steps:
1. Push your code to a GitHub repository.
2. Go to [vercel.com/new](https://vercel.com/new) and import the repository.
3. Make sure to choose **Vite** as the framework preset.
4. Click **Deploy**. Vercel will handle the routing and serve `dist/public` successfully thanks to the built-in rewrite configuration.

## ЁЯТ╝ Featured Work Implementations

A major highlight of this portfolio is the embedded **Scrubbable Modal Project Components**. 

Rather than static images, massive, long-form vertical screenshots of deployed systems (e.g., *Money Guard*, *Cinemania*) are sliced or combined programmatically. Inside the project modals, they act as an integrated browser window with native `overflow-y-auto` functionality.

This provides the viewer with exactly what feeling an app has without them ever having to open an external link.
